# Breeze Socialite Auth

> Starter kit de autenticação Laravel com Breeze e Socialite, pronto para login social e gerenciamento de usuários.

## Sobre o projeto

Este projeto combina o **Laravel Breeze** e o **Laravel Socialite** para fornecer um sistema de autenticação moderno, leve e extensível. Ele serve como base para desenvolvedores que desejam implementar autenticação com **Google, GitHub e outros provedores sociais**, mantendo o sistema nativo de Breeze para registro, login, recuperação de senha e 2FA.  

O foco principal é fornecer um **esqueleto de autenticação completo**, permitindo que outros desenvolvedores adicionem facilmente novos provedores sociais e recursos avançados de segurança.

### Recursos incluídos

- Autenticação nativa Laravel Breeze (login, registro, recuperação de senha, 2FA)
- Autenticação social com Socialite (Google, GitHub, Facebook)
- Criação automática de usuários ao logar via provedores
- Integração completa com rotas, controladores e views
- Estrutura organizada para facilitar a expansão
- Componentes Blade prontos para login social
- Preparado para testes locais com OAuth de desenvolvimento

### Próximos recursos planejados

- Adição de mais provedores sociais (Facebook, LinkedIn, Twitter)
- Relacionamento entre usuários Socialite e usuários internos
- Migração de usuários para contas completas
- Melhorias no front-end com componentes Blade reutilizáveis
- Documentação completa para desenvolvedores

## Tecnologias

- **Laravel 10**
- **Laravel Breeze**
- **Laravel Socialite**
- **TailwindCSS**
- PHP 8.x
